   🐍 Snake Water Gun Game (Python)

A fun and interactive command-line implementation of the classic **Snake–Water–Gun** game built using Python. This project demonstrates core programming concepts like loops, conditionals, dictionaries, and user interaction.

---

## 🎮 Game Overview

Snake–Water–Gun is similar to Rock–Paper–Scissors, where each choice defeats one and loses to another:

* 🐍 Snake drinks Water → Snake wins
* 💧 Water douses Gun → Water wins
* 🔫 Gun kills Snake → Gun wins

---

## ✨ Features

* 🔁 **Multiple Rounds Gameplay** (5 rounds per game)
* 🎯 **Score Tracking System**
* 🎲 **Random Computer Moves**
* ⚠️ **Input Validation**
* 🧠 **Dictionary-based Mapping for Clean Logic**
* 💬 **User-friendly Command Line Interface**

---

## 🛠️ Tech Stack

* Python (Core concepts only)
* No external libraries required

## 🚀 How to Run

1. Clone this repository:

   ```bash
   git clone <your-repo-link>
   ```

2. Navigate to the project folder:

   ```bash
   cd snake-water-gun
   ```

3. Run the Python file:

   ```bash
   python main.py
   ```

---

## 🧩 How It Works

* The user inputs one of: `snake`, `water`, or `gun`
* The computer randomly selects a choice
* The winner is decided based on predefined rules
* Scores are updated after each round
* Final winner is घोषित after 5 rounds

---

## 📌 Example Gameplay

```
Round 1
Enter your choice: snake
Computer chose: water
You chose: snake
You win!

Score → You: 1 | Computer: 0
```
